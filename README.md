##研究活動  
２つの研究テーマに取り組んでおり、それぞれ国際会議論文を投稿しています。  
１つ目のテーマが「ルールマイニングのための進化計算に対する特徴選択の導入」です。従来の大規模データにおけるルールマイニングには時間がかかる上、冗長なルールが生成されるという問題がありました。そこでルール発見と同時に特徴選択を実行することで、短時間でユーザーが必要とするだけのルール数を得ることができるようになりました。  
こちらは進化計算分野のトップ国際会議GECCO2025にてポスター論文として採択されました。  
  
２つ目のテーマが「局所分布ルールを用いた回帰に対する説明可能な特徴選択手法」です。従来の特徴選択の一種であるフィルタ法は回帰に対する研究が限定的でした。また、特徴量の選択理由の説明が不足してました。そこで、ある目的変数の局所的な分布を特徴づけるような特徴量を特定することで回帰に対する特徴選択を実現しました。また、ルールの理解しやすさを活かすことで特徴量の選択過程の説明性を高めることを可能にしました。こちらの論文は現在査読中です。
